# AI Backgammon 🎲

A fully playable, graphical Backgammon game featuring a custom AI opponent trained using Genetic Algorithms!

## 🎮 How to Play (For Gamers)
You do not need to download Python or know how to code to play this game! 
1. Look on the right side of this GitHub page and click on **Releases**.
2. Download the latest `AI_Backgammon.exe` file.
3. Double-click the file to launch the game and play against Adam!

---

## 💻 Developer Setup (For Coders)
If you want to read the code, tweak the AI's Genetic Algorithm, or run the Python scripts manually:

1. Clone this repository to your local machine.
2. Open your terminal in the project folder and install the dependencies:
   ```bash
   pip install -r requirements.txt
3. Run the main game loop:
    python client/gui_main.py
